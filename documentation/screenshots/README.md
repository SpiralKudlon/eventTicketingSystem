## üìù Tips for Screenshots

1. **Use browser dev tools** to capture consistent screenshots
2. **Crop to remove browser chrome** for cleaner images
3. **Use PNG format** for UI screenshots (better quality)
4. **Use JPG format** for photos (smaller file size)
5. **Consider using African-themed stock photos** from:
   - [Unsplash Africa Collection](https://unsplash.com/s/photos/africa-event)
   - [Pexels Africa](https://www.pexels.com/search/african%20festival/)
   - [Pixabay Kenya](https://pixabay.com/images/search/kenya%20festival/)
