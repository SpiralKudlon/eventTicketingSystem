## How to Capture Screenshots

### Event Browsing
1. Start both backend and frontend servers
2. Navigate to `http://localhost:3000`
3. Wait for events to load
4. Take a full-page screenshot showing multiple events

### Ticket QR Code
1. Click "Buy Ticket" on any event
2. Fill in the checkout form with test data:
   - Name: John Kamau
   - Email: john.kamau@example.com
   - Phone: +254700123456
   - Quantity: 2
3. Click "Complete Purchase"
4. Take a screenshot of the confirmation page with QR code

## Screenshot Guidelines

- Use high resolution (at least 1920x1080)
- Ensure all text is readable
- Show the full browser window
- Include the URL bar to show localhost
- Capture the complete user interface

## Tools for Screenshots

- **Linux:** GNOME Screenshot, Flameshot, Spectacle
- **Windows:** Snipping Tool, Snip & Sketch
- **Mac:** Command + Shift + 4
- **Browser:** Browser DevTools screenshot feature
